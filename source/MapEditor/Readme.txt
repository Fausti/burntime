Import/Export of original Burntime Raw maps:
 What you can do:
 - change size of a map
 - change tiles of the same set
 What you can't do:
 - add new tiles
 - change the count of entrances
 What possibly will not work:
 - using tiles of other tile sets
   e.g. changing Sana into a factory themed map
